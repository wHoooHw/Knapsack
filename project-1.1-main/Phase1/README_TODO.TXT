PROJECT TITLE: 
BCS1300 Project 1.1 - Pentominoes

PURPOSE OF PROJECT:
To find out if a given sized rectangle can be fully covered with Pentominoes and if yes we should provide one solution. 

VERSION or DATE:
12.10.2023

HOW TO START THIS PROJECT:
Add the remaining Pentominoes to the PentominoBuilder file, resolve the issue with colision and out of bounce. Include a checking mechanism to check if the field is full.
